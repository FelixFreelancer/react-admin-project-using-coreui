## Installation

``` bash
# clone the repo
$ git clone https://github.com/FelixFreelancer/react-admin-project-using-coreui

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

### Basic usage

``` bash
# dev server  with hot reload at http://localhost:3000
$ npm start
```
